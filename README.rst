=======
Jsonify
=======

Example bot for developers.airslate.com .

Usage
=====

**Build images:**

.. code-block:: bash

   $ make build


**Run application:**

.. code-block:: bash

   $ make up

**List containers:**

.. code-block:: bash

   $ make ps

**Uninstall:**

.. code-block:: bash

   $ make dist-clean
   $ docker image prune # optional
